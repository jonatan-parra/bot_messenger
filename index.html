Bot de Messenger

Utilizando las herramientas de Facebook Developers para la implementación de bots que interactuan en Messenger se diseñó un bot para una pastelería

Miralo funcionar en este video

En el video se muestra la interacción entre el bot y el usuario

El bot está diseñado para una pastelería la cual quiere publicitar y atender a sus clientes a través de Facebook Messenger

Disfruta de:

  • Saludo personalizado
  • Mensaje inicial
  • Menú anidado
  • Despliegue de imágenes y texto
  • Respuesta en texto
  • Envio del imágenes del bot al usuario
  • Llamadas
  • Visualización de mapas
  • Enlaces a páginas externas




Explicación de las funciones

Entrar al chat

En el momento en que el usuario entra a chat de bot este lo saluda con el nombre del usuario. En este caso Jonatan. Para que este seleccion la opción Empezar

Inicio chat

En el momento que el usuario presiona el boton de Empezar , el bot le manda un mensaje inicial indicandole que interactue a través del menú del chat

Menú principal

En el menú principal se pueden tener varias secciones que conducen a su respectivo submenú. En este caso las opciones son: Menú de productos, información de compra e información de contacto


Submenú

Al ingresar en la opción Menú de productos se muestra el respectivo submenú, el cual contiene las opciones de motrar pasteles o cupcakes

Mostrar pasteles

Cuando se selecciona la opción pasteles el bot trae la lista de pasteles disponibles, cada elemento contiene: una imagen, titulo y descripción

Respuestas de texto

Dentro del menú Información de compra en el momento en que el usuario selecciona la opción Forma de pago el bot trae una respuesta en texto y se la muestra al usuario


Respuesta de imagen

Cuando el usuario selecciona Promoción mensual el bot trae una respuesta en texto e imagen

Realizar llamada

Al seleccionar la opción Llámanos el usuario puede seleccionar el boton que le aparece en pantalla, lo cual abrirá la aplicación de llamadas y le permitirá contactarse con un asesor

Respuestas en texto

Dentro del menú Información de compra en el momento en que el usuario selecciona la opción Forma de pago el bot trae una respuesta en texto y se la muestra al usuario


Mapas

Cuando el usuario selecciona Ubica nuestro local el bot trae una lista de ubicaciones de Google Mapas en las cuales el usuario puede explorar y desplazarse sin salir de la apliación de Messenger, como se muestra en el video inicial
